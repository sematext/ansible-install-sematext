Ansible Sematext SPM Monitor Install Role
=========================================

Install Sematext SPM Monitor.

Dependencies
------------
None

Example Playbooks
-------------------------
```
- hosts: servers
  sudo: yes
  roles:
    - { role: sematext.spm-monitor-install}
```
